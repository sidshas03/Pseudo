What is PseudoJSON?

PseudoJSON is a free REST API developed as a small project. It was created as a helper for when you need dummy data for a quick testing and prototyping of your Frontend applications.

Try it Out!

Example Code

              
  fetch('https://pseudojson.onrender.com/quotes/1')
  .then(res => res.json())
  .then(json => console.log(json));

  //fetch data to get the output

Resources

Here are the resources that can be accessed through our API:

/quotes 100 quotes
/posts 100 posts
/users 20 users

Note: Results are paginated by 20. Pass pageSize=100 as a query to get all results. Also, new resources may be added in the future.

Routes

HTTPS and HTTP are both supported.

GET /quotes Get all quotes
GET /quotes/1 Get a quote by ID
GET /quotes?category=success Get quote by category
POST /quotes Add a quote
PATCH /quotes/1 Modify a quote
PUT /quotes/1 Replace a quote
DELETE /quotes/1 Delete a quote

Note: You can take a look at the documentation for usage examples.
